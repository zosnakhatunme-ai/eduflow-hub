Setup in 5 Minutes

Get your own LMS running with minimal configuration.

1

Create Firebase Project

Go to Firebase Console, create a project, enable Authentication (Email/Password) and Firestore Database.

2

Update Firebase Config

Replace the config in src/lib/firebase.ts with your Firebase project credentials.

3

Set ImgBB API Key

Get a free API key from imgbb.com and update it in src/lib/firebase.ts.

4

Configure App Settings

Login as admin, go to Settings → change app name, logo, payment methods, and social links.

5

Deploy

Deploy to Netlify, Vercel, or any static hosting. Your LMS is live!
